Sumin Leem 2024

This repository contains magma files computing the examples in "Fast Divisor arithmetic in the divisor class group" by J. Bauch, H. Tran, and S. Leem. To run the examples, you may need to download all 5 files. This code is with Magma V2.27-5 or later version. Since this examples are small, Magma online calculator (http://magma.maths.usyd.edu.au/calc/) can be used to run them.
